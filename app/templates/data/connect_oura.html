Instructions to get your Oura Personal Access Token
  1. Log in to the Oura Cloud Dashboard
  2. Navigate to the Personal Access Tokens section
  3. Click on Create a New Personal Access Token
  4. Enter a name for your token (e.g. "Health Tracker App")
  5. Select the following scopes:
    • daily
    • heartrate
    • personal
    • session
    • workout
  6. Copy the generated token
  7. Paste it in the form below
Your token will be stored securely and used only to access your Oura data.
